API 接口说明

获取RSS数据

接口：/api/rss

说明：获取所有RSS源的最新文章，返回JSON格式数据

手动刷新RSS

接口：/api/refresh

说明：手动触发RSS数据更新，需要POST请求

修改更新间隔

接口：/api/update-interval

说明：设置自动更新间隔，需要POST请求，参数格式：

{"interval": 分钟数}

数据文件位置

RSS源列表：rss_feeds.json

RSS数据文件：rss.json
